== Installation ==

1. Upload the 'ai-magic-content-generator' folder to the '/wp-content/plugins/' directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Go to the AI Magic Content Generator settings page to configure your preferences.
4. Start generating content by selecting the 'Create New Post with AI Magic' option within the post editor.
